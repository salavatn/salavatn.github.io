FILTER

  1. Query: python table = TableCustomers query = session.query(table).order_by(table.name).filter(table.age <= 30)

  2. Get data: ```python for get in query: print(f"\tget.name: {get.name}, \tget.age: {get.age}, \tget.grade: {get.grade}")

    output get.name: Jack, get.age: 21, get.grade: A get.name: Marry, get.age: 26, get.grade: R ```

  3. LOG: SQL Format: sql SELECT "Customers".id AS "Customers_id", "Customers".name AS "Customers_name", "Customers".age AS "Customers_age", "Customers".grade AS "Customers_grade" FROM "Customers" WHERE "Customers".age <= %(age_1)s ORDER BY "Customers".name
